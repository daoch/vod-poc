## ğŸš€ Getting Started

### Prerequisites

- .NET SDK
- Node.js
- npm

### Installation

1. Clone the repository: 
```bash
git clone https://github.com/daoch/vod-poc.git
cd vod-poc
```

2. Install backend dependencies:
```bash
cd backend
dotnet restore
```

3. Install frontend dependencies:
```bash
cd frontend
npm install
```

### Running the Application

1. Start the backend server:
```bash
cd backend
dotnet run
```

2. Start the frontend development server:
```bash
cd frontend
npm start
```

3. Open your browser and navigate to `http://localhost:3000` (or the configured port)
